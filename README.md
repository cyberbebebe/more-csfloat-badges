## ✨ Features:

A browser extension for [CSFloat](https://csfloat.com) that expands the pattern-based badge system.

1. **Comprehensive Pattern Badges:** Injects expanded tier badges directly into item cards (see screenshots below) in the most native way. This fills the gap left by native CSFloat badges (which cover only from the 1st to 3rd Fire and Ice tiers) and BetterFloat badges (which covers only few items):
   - **Marble Fade Knives:** From 4th Max down to Fake Fire & Ice (from 1st for Talon).
   - **Fade Gloves:** Ranks (1st-3rd, above tiers), Tiers (1st-3rd, BTA), and the Orange pattern.
   - **Gamma Doppler (Phase 2):** Max Green tiers 1st through 4th, plus BTA (Better Than Average) patterns.
   - **Support for more patterns in the future (like max pink and max red tips, etc.)**

2. **Enhanced Sales History Table:** Upgrades the UI and convenience of the "Latest Sales" table. It automatically hides the "Stickers" column for knives and integrates pattern badges directly into the sales rows, making price-checking rare patterns much faster.

3. **BetterFloat Synergy:** Fully compatible with the [BetterFloat extension](https://github.com/GODrums/BetterFloat). It works exceptionally well with their "Show In-Game Pictures" feature, allowing you to **instantly compare the visual differences** between pattern tiers directly on the item cards.

> To prevent UI clutter and duplication, this extension excludes certain badges that are already handled by BetterFloat (see the "Excluded Patterns" table below).

## 🛠️ Install:

1. Download the latest extension .zip from the [Releases page](https://github.com/cyberbebebe/more-csfloat-badges/releases)
2. Unzip the archive to a folder on your computer
3. Open Chromium-based browser (Chrome, Brave, Edge, Opera, etc.):
   - Go to `chrome://extensions/` (or usually "Menu -> Extensions -> Manage Extensions").
   - Enable **Developer mode** (toggle in the top right corner).
   - Click **Load unpacked**.
   - Select the folder where you unzipped the extension.
4. Reload the CSFloat page

## Excluded Patterns (Fully or partially covered by BetterFloat)

To ensure perfect compatibility and avoid duplicate badges, the following patterns are intentionally left to be handled by BetterFloat:
| Skin | Patterns |
| --- | --- |
| Doppler | Max Ruby / Max Sapphire |
| Gamma Doppler | Phase 1 Diamond / Emerald |
| Nocts | Max Black tiers |
| Overprint | Arrow / Flower / Mixed / Polygon patterns |
| Crimson web, Emerald web | Single / Double / Triple web tiers |
| Crimson Kimono | Max Red tiers |

> This list is subject to change as I'm discovering which items BetterFloat covers.
